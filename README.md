# ipon 💰
> mag-ipon ng pera — save your money

A mobile-first savings goal tracker built with React + Vite + Supabase.

## Features
- Anonymous usage — no sign-up required to start
- Progressive auth — sign up to sync across devices and share goals
- AI-suggested goal icons
- Light / dark mode
- Real-time Supabase backend

## Local Development

```bash
npm install
npm run dev
```

## Deploy to Vercel
1. Push this repo to GitHub
2. Import the repo on vercel.com
3. Vercel auto-detects Vite — just click Deploy

## Stack
- React 18
- Vite 5
- Supabase (auth + database)
